# Chuski Templating Starter Kit

With this starter kit you can quickly start coding your project. 
It's based on Vite, SCSS, and JavaScript. 
You can add all the external libraries you need.
Enjoy coding!

### Install npm packages
```bash
npm install
```

### Run dev server
```bash
npm run dev
``` 

### Build project
```bash
npm run build
```
